Filter Element For Oxygen Builder

== Description ==
Custom Element for Filtering Post in Repeater Element Or Easy Post Element in Oxygen Builder

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
3. Go to Oxygen Builder
4. Create Repeater Element
5. Select Oxywhello Filter
6. Insert Container Class Repeater Element
7. Insert Class for item Repeater Element
8. Select what you want to filter (ex, category, taxonomy, or post tag)
